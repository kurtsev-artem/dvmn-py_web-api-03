# Обрезка ссылок с помощью Битли

Приложение предназаначено для сокращения ссылок при помощи API VK и подсчёта количества переходов по сокращённым ссылкам

### Как установить

Для получения токена доступа необходимо зарегистророваться на vc.ru и создать приложение. Токен разместить в фойле .env

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

### Цель проекта

Код написан в образовательных целях на онлайн-курсе для веб-разработчиков [dvmn.org](https://dvmn.org/).
